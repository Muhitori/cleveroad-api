## Installation

```bash
$ npm install
```

## Migrations

Run npm build to create dist folder!
App use compiled migrations.

```bash
$ npm build
$ npm run typeorm migration:run
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
